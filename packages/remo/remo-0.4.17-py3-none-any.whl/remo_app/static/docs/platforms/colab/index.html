    Remo on Google Colab¶

    Google Colab allows to use Google Drive as data storage, otherwise your data will be lost.

    Remo can be used with both options:

    • with storing data and backuping database on Google Drive
    • with storing data on disk (data will be lost as your Colab session finished)

    Mount Google Drive (optional step)¶

    Do the following steps to mount your Google Drive at /gdrive location: 

    from google.colab import drive
drive.mount("/gdrive")

    Installation¶

    Do the following steps to install remo: 

    !pip install remo
!python -m remo_app init --colab

    There is optional parameter --remo-home, which allows to set different location for remo home directory. Default location is /gdrive/My Drive/RemoApp

    If you previously have used remo on Colab and backuped data on Google Drive, your database will be restored from backup.

    Use remo¶

    After installation completed, you can use remo directly from Colab Notebook: 

    import remo
remo.open_ui()

    Backup (optional step)¶

    When you use Colab with mounted Google Drive, remo images already stored on Google Drive and database each minutes do automatic backup. If you want to be sure that latest changes backuped, run the following command: 

    !python -m remo_app backup

    Backup files can be found at <remo_home_dir>/backup/
